# Excecuter dans le cadre du developpement

1. Cloner le projet
2. Executer la commande suivante:

```
npm install
```

3. Installer Nest CLI

```
npm i -g @nestjs/cli
```

4. Base de données

```
docker-compose up -d
```

## Pile utilisée:

- MongoDB
- NestJS
